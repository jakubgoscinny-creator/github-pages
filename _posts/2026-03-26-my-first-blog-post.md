## What I Learned

- How to enable **GitHub Pages** on a repository
- How to configure a Jekyll site with `_config.yml`
- How to customize a homepage with `index.md`
- How to write blog posts using Jekyll frontmatter

## What is GitHub Pages?

GitHub Pages is a free static site hosting service that publishes any static files directly from a GitHub repository. It supports **Jekyll** out of the box, which means you can write content in Markdown and it will be automatically converted into a beautiful website.
